# 🐝 시연공주와 꿀벌들

**개발자 중심 협업 웹 플랫폼**  
GitHub의 단점 보완 + Discord의 실시간 피드백 장점을 융합한  
**개발 협업을 위한 그룹 스페이스 웹**

---

## 📌 주요 기능

- 🔐 **회원가입 / 로그인** (JWT 기반 인증)
- 👥 **그룹 생성 및 참여 (초대코드 시스템)**
- 📁 **파일 업로드 및 버전 관리**
- 💬 **그룹 댓글 시스템**
- 📣 **알림 기능**
- ⚙️ **마이페이지: 비밀번호 변경, 탈퇴 등**
